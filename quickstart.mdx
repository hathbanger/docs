### Step 2: Configure your wallet ğŸ‘›

Run the setup wizard to configure your private key and network:

```bash

```

You'll be prompted to:
1. Enter your private key (or generate a new one) ğŸ”‘
2. Select network: Base Sepolia (testnet) or Base (mainnet) ğŸŒ
3. Set your preferences (ASCII art, etc.) ğŸ¨

### Step 3: Fund your wallet ğŸ’°

<Tabs>
  <Tab title="Base Sepolia (Testnet)">
    Get free testnet tokens:
    1. Visit [Base Sepolia Faucet](https://faucet.quicknode.com/base/sepolia)

  <Tab title="Base (Mainnet)">

Check your balance:

```bash

```

### Step 4: Create your first token ğŸª™

```bash
httpcat create "My Token" "MTK" --website https://mytoken.io
```

This costs $0.01 USDC and returns your token address (ending in `402`). ğŸ¯

<Tip>
Save the token address! You'll need it for trading. ğŸ’¾
</Tip>

### Step 5: Trade tokens ğŸ“Š

Buy tokens:

```bash
# Testnet amounts: 0.05, 0.10, 0.20 USDC
httpcat buy 0x1234...402 0.10

# Mainnet amounts: 50, 100, 200 USDC
httpcat buy 0x1234...402 100
```

Sell tokens:

```bash
# Sell 50% of your position
httpcat sell 0x1234...402 50%

# Sell all
httpcat sell 0x1234...402 all
```

Check your positions:

```bash
httpcat positions
```

## Beyond Token Trading ğŸŒŸ

The launchpad is just the beginning. httpcat-cli also supports:

### Universal Swaps

Swap any ERC20 token with DEX aggregation and best price routing:

```bash
# Swap tokens with automatic best price routing
httpcat swap 0xTokenIn 0xTokenOut 1000000

# Works with any token, not just 402.cat tokens
```

### Agent Mode

Natural language interface for any operation:

```bash
httpcat agent "What are the top tokens by market cap?"
httpcat agent "Buy 0.10 of the newest token"
```

### MCP Integration

Expose all operations to Claude, Cursor, or any MCP-compatible AI assistant:

```bash
httpcat mcp-server
```

See the [MCP documentation](/mcp/overview) for setup details.
