## What is 402.cat?

402.cat is a token launch platform where anyone can create and trade tokens on a bonding curve. Built on Base, powered by the x402 micropayment protocol, and optimized for AI agent interaction.

  <Card
    title="Create Tokens"

  </Card>
  <Card

  </Card>
  <Card

  </Card>
  <Card

  </Card>
